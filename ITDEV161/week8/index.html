Your Location

This example will find your location and display it on the map.

Tutorial is at Google Geolocation Map Tutorial

This file contains logic and API errors. Start by getting your own Google API key (use yours, not mine!) and using it in the script tag at the bottom. When you get the map working, add a Marker (google.maps.Marker object) (info here) for the user's current location. Use a tooltip on the marker that says something like "Here I am!". Have fun finding and fixing the errors for part of assignment 8.
